Aethvion Suite

View Source on GitHub


A unified AI-powered super-app ecosystem designed for versatility and power. From advanced multi-step agents to professional audio editing and code execution, Aethvion Suite brings together the next generation of creative and productivity tools.

Agent Workspace
Multi-step ReAct agent runner with real-time SSE event streaming for complex task automation.
Agent Corp
Manage and collaborate with multiple persistent specialized agents in a unified corporate infrastructure.
Misaka Cipher
Multi-provider AI chat with threads, auto-routing, and context modes for the ultimate conversational experience.
Aethvion Photo
AI image generation and precision layer-based editing system for digital artists and creators.
Aethvion Audio
Professional multi-track timeline editor with live waveforms and real-time effects processing.
Aethvion Code IDE
Monaco-based IDE with integrated AI copilot and secure sandboxed code execution environment.
Direct Local Inference
Run GGUF models (Mistral, LLaMA) directly on your hardware for privacy and performance.
Aethvion Drive Info
Recursive system storage analysis and visual representation of your digital space.

Usage & Cost Tracking

Monitor token usage, cost estimates, and granular per-query breakdowns to keep your AI operations efficient and transparent.
